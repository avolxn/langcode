## Этап 1: Анализ и проектирование

**Цель:** Понять архитектуру OpenCode и спроектировать Python реализацию.

### Задачи

1. **Анализ OpenCode** - исследовать `./opencode/` для понимания:
   - Высокоуровневая архитектура и ключевые модули
   - Точки входа и поток выполнения
   - Система инструментов (регистрация, выполнение)
   - Система промптов (хранение, шаблонизация, композиция)
   - Управление состоянием и память
   - Интеграция MCP/LSP
   - Конфигурация и настройки
   - Что можно пропустить (desktop/web UI)

2. **Проектирование Python реализации:**
   - Структура проекта и организация модулей
   - Ключевые дизайн-решения
   - Обработка различий TypeScript → Python
   - Соответствие зависимостей npm → Python
